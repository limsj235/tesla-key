나는 테슬라 차량 소유자로 테슬라 API를 활용해 개인용 안드로이드 핸드폰용 앱을 만들고 싶어 앱 이름은 쭈삼 으로 하고
Tasker + Bolt for Tesla 참고하여 삼성의 빅스비 음성명령으로 테슬라 차량을 쭈삼으로 직접 제어할수 있도록 개발 하고싶어
주요 기능
- 상단에 메뉴 : 설정, 테슬라 제어, 
- 명령 실행 시 슬립모드이면 차량 깨우기
-  테슬라 제어 화면 및 음성 명령 가능 처리
	1. 트렁크 열고 닫기
	2. 프렁크 열기
	3. 윈도우 환기(열고 닫기)
	4. 정차시 조수석 문열기
	5. 비상등 켜고 끄기
- 기타기능
	1. 블랙박스 영상 보기 (클립 시간대별 전방, 후장, 양쪽 측면)
		- 핸드폰의 저장소 또는 usb로 연결 하여 폴더 선택 하여 영상 볼수 있게
		- 영상은 전방 후방 측면등을 같은 시간데 클립을 모두 한번에 볼수 있게
		- 위의 영상을 하나의 영상으로 시청 가능 하도록 하나의 동영상으로 생성 되도록 구성
		- 
		
	
- 접속 방법은 사용자 인증은 테슬라 차량 앱을 동해 인증 
- 앱 실행시 아규먼트를 받아 테슬라 제어 가능 처리
- 추후 기능 확장성을 고려
